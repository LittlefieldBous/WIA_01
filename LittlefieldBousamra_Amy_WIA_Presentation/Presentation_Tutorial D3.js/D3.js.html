Learning D3.js

What is D3.js?

D3.js is a javaScript library used for manipulating data

D3.js was created by Mike Bostock during his doctorate studies at Stanford.

What is D3.js used for?

D3.js is used for displaying many forms of data.

What are the advantages of using D3.js?

When a developer needs users to interact with data on a webpage D3.js allows designers to build a data visualization framework that fits their needs.

Other advantages of using D3.js:

  • Allows the designer to create transitions & animations
  • Includes built in physics that allows for gravity and friction
  • Provides click and drag behaviors

Let's take a closer look at the concept of Selection in D3.js

  • This selects an element from the DOM in order to manipulate it in some way.
  • Then it appends it to svg to create an svg container where the chart will be placed.

    • For Example: In D3.js the commands d3select(#chart) targets just the chart inside the DOM. In contrast the command d3selectAll('p')would target all the p tag elements. This concept is similar to JQuery selection: For example $('body') which selects the body element.

    
var myChart = d3.select('#chart').append('svg')

    Animate separate pieces of the pie!

    For fun can add a little extra to make it interactive

    
.on('mouseover', function (d, i) { //create a mouseover function
 d3.select(this) // selects slice
.transition()//calls transition function/method
.duration(500)//sets time
.ease('elastic') //sets animation effect
.attr('transform', function (d) { //transform function..transform moves a shape
var dist = 10; //how far the slice will go out
d.midAngle = ((d.endAngle - d.startAngle) / 2) + d.startAngle; //the data objects angle math to move angle outward
var x = Math.sin(d.midAngle) * dist; //math that moves the angle's x axis
var y = -Math.cos(d.midAngle) * dist; //math that moves the y axis angle out
return 'translate(' + x + ',' + y + ')'; //moves the shape coordinates x and y.
  });
}).on('mouseout', function (d, i) {
d3.select(this) //refers to slice
.transition() //calls transition
.duration(500)//time after mouse leaves
.ease('bounce') //applying animation bounce effect
.attr('transform', 'translate(0,0)') //moves shape back to original coordinates.
});
